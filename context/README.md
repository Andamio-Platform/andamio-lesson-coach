# Context Directory

This directory contains background materials and guidelines that inform the lesson coach.

## Files to Add

### sample-slts.md
Real Student Learning Targets from existing Andamio courses. Use these to test coach output.

### module-examples.md
Complete Module structures showing SLTs + Assignments. Helps coach understand how lessons fit into the larger context.

### language-guide.md
Examples of contribution-centered language vs LMS language. Ensures coach maintains Andamio's framing.

### platform-docs/
Relevant Andamio Platform documentation, especially:
- Course/Module structure
- How credentials work
- Project contribution workflows
- Current Platform features (for Product Demo lessons)

### partnership-examples/
Sample courses from:
- Lido Nation
- FC Barcelona
- Farmer courses
- Any other partnerships

Shows real-world usage and quality standards.
