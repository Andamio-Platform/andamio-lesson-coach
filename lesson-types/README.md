# Lesson Types

This directory contains the coach prompts for each lesson type.

Each lesson type will have its own file defining:
1. Purpose and use cases
2. Input requirements
3. Output structure
4. Pedagogical approach
5. Contribution-centered framing
6. Example coach prompt
